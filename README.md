# #smokefleet avatar creator

A premium web application to design, edit, and customize original pilot avatars.

## Features
* **Perspective Overlays**: Layered controls for shades, joints, and backgrounds with custom scaling and rotation.
* **Satirical Watermarks**: Customizable typography presets (like *Stable Genius*, *Fake News*, *MAGA Tears*) with adjustable fonts, drop shadows, and vibrant glows.
* **Eraser Workspace**: Responsive brush size preview coordinates on hover for precision asset masking.
* **Procedural Smoke Plumes**: Realistic particle-generated smoke trails with customizable dispersion, speed, and color swatches.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
